How these repos relate

The MJ monorepo is the primary platform. Skyway, Forge, and the VSCode extension are independent tools that depend on MJ packages but ship on their own cadence. The BizApps repos are reusable application building blocks consumed by end-user MJ apps. Sample apps demonstrate end-to-end deployment patterns.

Each repo is its own world: separate releases, separate issues, separate Discussions where relevant. This page is a directory, not a translation layer.
