Contract

Input: ListDelta object from ListOperations.computeDelta() (or any compute* method) including deltaToken (HMAC of source/target/timestamp).
Output: emits confirm with the validated deltaToken, or cancel.
Drop guard: when counts.remove > 0 the confirm button is disabled until the acknowledgment checkbox is checked. applyListDelta on the server also rejects the call when confirmDrops !== true — the UI is a convenience, not the source of truth.
Staleness: tokens expire after 5 minutes. If records changed since preview, server returns STALE_DELTA and the component re-runs computeDelta automatically.
Used by: Refresh-from-source, Set-op into existing list, Bulk-move records, any future "replace contents" op.
