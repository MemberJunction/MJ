Beyond the core

The ecosystem

Subprojects, BizApps, integrations, deployment tools, and developer ergonomics — everything that ships alongside MemberJunction core.

BizApps — packaged business verticals

CRM

Accounts, contacts, opportunities, pipeline management, activity tracking. Drop-in entity extensions and Actions.

@memberjunction/actions-bizapps-crm

Accounting

Chart of accounts, AR/AP, journal entries, period closing, financial reporting integrations.

@memberjunction/actions-bizapps-accounting

LMS

Courses, enrollments, certifications, progress tracking. Integrates with the content autotagging pipeline.

@memberjunction/actions-bizapps-lms

Social / Community

Channels, posts, threading, reactions, member feeds. Designed for association and member-org communities.

@memberjunction/actions-bizapps-social

Form Builders

Event registration, membership renewal, custom data capture forms. Generates entities + UI from form definitions.

@memberjunction/actions-bizapps-formbuilders

Knowledge Hub

Content autotagging, taxonomy management, semantic search, RAG+ context — the AI-native KMS layer.

@memberjunction/ai-knowledge

Deployment & ops

Docker Images

Production MJAPI container published on every release. Plus a Docker Workbench for sandboxed development against a dedicated SQL Server instance.

Hosted MJ (MJC)

MemberJunction Central — fully hosted MJ deployment. Sign up, point at your data, skip the ops entirely.

Visit MJC ↗

OpenApp

The deployment format for packaging Skip apps + brain + entity extensions as one publishable unit. Versioned, no-breaking-changes governed.
