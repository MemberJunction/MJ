What this shows

  • RAG badge — collapsible panel above the first assistant reply showing the scopes + results the agent saw before generating.
  • Per-scope grouping — each scope block is expandable; each result shows score, source, entity, tags.
  • Tool-call surfaces — mid-turn Scoped Search invocations get their own styled strip so the user can trace the agent's reasoning.
  • Open as artifact — one-click persists the full result set to the artifact system (using the `Search Result Set` artifact type).
  • No hardcoded colors — the RAG badge uses --mj-status-info-* tokens; dark mode inherits automatically.
