What this shows

  1. Scope chip — sits to the left of the search input. Shows current selection (Global, one scope, or N scopes).
  2. Dropdown — groups scopes by visibility: Default (Global), Organization (shared), Personal (owner-only).
  3. Time-windowed scopes — show an inline badge noting when they're active (e.g., Q4 Financial Reports).
  4. Default scope — one org scope per user is marked "Default" (via IsDefault=1) and auto-selected.
  5. Global reset — picking Global clears all other selections; selecting any specific scope removes Global.
  6. Design tokens only — all colors route through --mj-* variables so the dark-mode variant (mockup 11) is a zero-code change.
