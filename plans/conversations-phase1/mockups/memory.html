Memory

User-visible control over what the assistant remembers — project-scoped (per conversation folder) and global, plus an incognito / Temporary chat that neither reads nor writes memory. Three divergent models below.

Option A

"What I remember" side panel

What: A docked "What I remember" panel beside the chat. Memories are grouped by Project (folder) and Global, each row inline-editable and deletable with provenance ("Learned 3 days ago"). A project filter narrows the view; a Temporary-chat toggle opts out entirely.

Why world-class: Mirrors the mental model of best-in-class assistants — memory is a browsable, trustworthy ledger, not a black box. Project grouping maps 1:1 to MJ Projects, so scope is always legible.

Tradeoff: Persistent panel competes with chat for horizontal space; on narrow screens it must collapse to a drawer.

What: Memory surfaces in the flow. The moment the agent learns something, a subtle "I'll remember:" chip appears that the user can Keep / Discard, and re-scope (This project ↔ Global) before it's committed. A "Manage memories" link opens the full list; incognito is a visually distinct chat state.

Why world-class: Consent happens at the point of learning, when context is freshest — no after-the-fact auditing required. Provisional → Active maps onto the existing Agent Memory lifecycle: a chip is a Provisional note awaiting the user's Keep.

Tradeoff: Chips can feel noisy if the agent learns often; needs batching/quieting rules and a clear path to bulk-manage later.

What: A dedicated management surface — searchable, filterable by Project and lifecycle state (Active / Provisional), with at-a-glance counts and a prominent incognito toggle. Provisional rows offer a one-click confirm (✓) to promote them to Active.

Why world-class: Treats memory as first-class data the user owns and governs. Exposes the real server-side Agent Memory lifecycle (Active/Provisional, scoped by Project/User/Company) directly, giving power users full transparency and bulk control.

Tradeoff: A separate destination means memory review is opt-in — without inline nudges, many users may never visit it, so it pairs best with a lightweight in-chat entry point.
