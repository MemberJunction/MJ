CSS Classes Reference

Container:
.mj-dropdown - Dropdown container (add .open class to show menu)

Trigger:
.mj-dropdown-trigger - Button that opens menu
.mj-dropdown-caret - Chevron icon that rotates when open

Menu:
.mj-dropdown-menu - The dropdown panel
.mj-dropdown-menu-right - Right-align the menu

Items:
.mj-dropdown-item - Clickable menu item
.mj-dropdown-item-disabled - Disabled item
.mj-dropdown-item-danger - Destructive action styling
.mj-dropdown-divider - Horizontal separator
.mj-dropdown-header - Section header text

Split Button:
.mj-btn-split - Container for split button
.mj-btn-split-toggle - The dropdown trigger part

Icon Button:
.mj-btn-dropdown-icon - Icon-only trigger button

Select Sizes:
.mj-select-sm - Small select
.mj-select-lg - Large select
.mj-select-with-icon - Container for select with icon prefix
