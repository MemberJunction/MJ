Buttons

Button styles for primary actions, secondary options, and various states.

Brand Color Comparison

All brand button colors at a glance for easy comparison.

Primary Buttons

Use for main actions like "Save", "Submit", or "Create".

<button class="mj-btn mj-btn-primary">Primary Button</button>

Secondary Buttons

Use for secondary actions like "Cancel", "Back", or "Filter".

<button class="mj-btn mj-btn-secondary">Secondary</button>

Ghost Buttons

Use for tertiary actions or in toolbars where minimal visual weight is needed.

<button class="mj-btn mj-btn-ghost">Ghost</button>

Accent Buttons

Use for highlighted actions, important callouts, or promotional CTAs. Amber/warm color provides visual emphasis.

<button class="mj-btn mj-btn-accent">Upgrade Now</button>

Tertiary Buttons

Use for secondary highlighted actions or complementary CTAs. Cyan/teal color provides an alternative accent.

<button class="mj-btn mj-btn-tertiary">Learn More</button>

Danger Buttons

Use for destructive actions like "Delete", "Remove", or "Revoke".

<button class="mj-btn mj-btn-danger">Delete</button>
