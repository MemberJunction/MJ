Complete Header Bar Redesign

Separated concerns: navigation, view selection, view actions, data actions

Data Explorer Contacts

Modified
Design principles:
  • View management (save, save-as, revert) grouped together, appear only when changes exist
  • View mode (grid/cards/timeline) in its own toggle group
  • Data actions (new record, export) separated from view actions
  • Configure button clearly positioned as a gear/settings action
  • Clear breadcrumb for navigation context
  • Record count shows total and filtered counts
