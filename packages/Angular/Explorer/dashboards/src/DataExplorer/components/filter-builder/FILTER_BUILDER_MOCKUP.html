MJ Filter Builder - Design Mockup

A modern, intuitive filter builder component for creating complex boolean filter expressions. Designed to be cleaner and more user-friendly than traditional query builders.

Design Notes: - AND/OR toggle at group level controls how conditions combine
- Visual connectors between rules show the logic clearly
- Nested groups allow complex expressions like (A AND B) OR (C AND D)
- Quick filter chips at top for common presets
- Generated SQL/filter shown at bottom for transparency
Boolean Field Example
Design Notes: Boolean fields get a special True/False toggle instead of a text input. This is more intuitive than typing "true" or "false".
Date Field Example
Design Notes: Date fields get a date picker. Special operators like "in the last 7 days" could show a number input + unit dropdown (days/weeks/months).

Alternative: Compact Read-Only View
WHERE Status equals "Active"
AND Amount greater than 1000
AND Created At is after Jan 1, 2024
Design Notes: This compact view could be used to display existing filters in a read-only summary. Click the edit button to expand into the full editor. Good for showing active filters in a toolbar or header area.
