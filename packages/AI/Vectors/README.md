# AI Vectors

This directory groups the MemberJunction packages for working with vector embeddings and vector databases. They enable semantic search, entity vectorization, duplicate detection, and synchronization between MemberJunction entities and vector stores.

## Packages

| Package | npm | Description |
|---------|-----|-------------|
| [Core](./Core/README.md) | `@memberjunction/ai-vectors` | Core vector operations and abstractions for entity vectorization |
| [Database](./Database/README.md) | `@memberjunction/ai-vectordb` | Vector database abstraction layer (index management, query operations) |
| [Dupe](./Dupe/README.md) | `@memberjunction/ai-vector-dupe` | Duplicate record detection using vector similarity |
| [Memory](./Memory/README.md) | `@memberjunction/ai-vectors-memory` | In-memory vector utilities |
| [Sync](./Sync/README.md) | `@memberjunction/ai-vector-sync` | Synchronization between MemberJunction entities and vector databases |
