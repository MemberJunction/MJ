# Example Manifest: MJAPI (Server-Side)

This is an example of the manifest generated by `mj codegen manifest` when run against the **MJAPI** application.

- **Dependencies walked:** 985
- **Packages with @RegisterClass:** 54
- **Total classes discovered:** 705

The manifest uses **named imports** for every `@RegisterClass` decorated class and places them all in an exported `CLASS_REGISTRATIONS` array, creating a static code path that prevents tree-shaking from removing them.

Notice the server-specific packages: AI providers (Anthropic, Azure, Bedrock, Gemini, Groq, etc.), actions (Apollo, CRM, accounting, social), communication providers (MS Graph, SendGrid), encryption, scheduling, storage, and more.
