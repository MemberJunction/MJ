# Business Application Actions

Action implementations for specific business application domains. Each package provides MemberJunction action wrappers for integrating with external business systems through the Actions framework.

## Packages

| Package | npm | Description |
|---------|-----|-------------|
| [Accounting](./Accounting/) | `@memberjunction/actions-bizapps-accounting` | Accounting system integration actions (QuickBooks, NetSuite, Sage, Dynamics) |
| [CRM](./CRM/) | `@memberjunction/actions-bizapps-crm` | CRM system integration actions (Salesforce, HubSpot, Dynamics, Pipedrive) |
| [FormBuilders](./FormBuilders/) | `@memberjunction/actions-bizapps-formbuilders` | Form builder and survey platform integration actions (Typeform, Google Forms, Jotform) |
| [LMS](./LMS/) | `@memberjunction/actions-bizapps-lms` | Learning management system integration actions (Moodle, Canvas, Blackboard, LearnDash) |
| [Social](./Social/) | `@memberjunction/actions-bizapps-social` | Social media integration actions (Twitter, LinkedIn, Facebook, Instagram, TikTok, YouTube, HootSuite, Buffer) |
