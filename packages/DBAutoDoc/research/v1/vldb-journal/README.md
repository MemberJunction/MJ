## Overleaf Upload Instructions

### Step 1: Create Overleaf Project
1. Open the **Springer Nature LaTeX Template** on Overleaf:
   https://www.overleaf.com/latex/templates/springer-nature-latex-template/gsvvftmrppwq
2. Click **"Open as Template"** to create a new project from it

### Step 2: Replace Template Files
1. Delete the template's sample files: `sn-article.tex`, `sn-bibliography.bib`, `sn-article.pdf`, `empty.eps`, `fig.eps`, `user-manual.pdf`
2. Upload `main.tex` and `references.bib` from this directory
3. Keep these template files (they are required):
   - `sn-jnl.cls` -- Springer Nature journal class
   - `sn-basic.bst` -- Basic numbered bibliography style
4. You can also delete the other `.bst` variants you won't use (`sn-nature.bst`, `sn-chicago.bst`, etc.)

### Step 3: Compile
1. Ensure the compiler is set to **pdfLaTeX** (Overleaf menu > Compiler)
2. Set the main document to **main.tex** (Overleaf menu > Main document)
3. Compile -- Overleaf runs pdflatex + bibtex automatically
4. Verify the PDF renders correctly

### Step 4: Generate .bbl for Editorial Manager
1. After successful compilation in Overleaf, download the `.bbl` file:
   - Click the **Logs and output files** button (next to Recompile)
   - Click **Other logs and files**
   - Download `main.bbl`
2. Alternatively, copy the content from the Overleaf file tree (it appears after compilation)
