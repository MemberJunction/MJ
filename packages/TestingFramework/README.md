# Testing Framework

Metadata-driven testing framework for MemberJunction supporting agent evals, workflow scenarios, and multi-oracle evaluation.

## Packages

| Package | npm | Description |
|---------|-----|-------------|
| [EngineBase](./EngineBase/) | `@memberjunction/testing-engine-base` | Metadata cache for test types, suites, and tests (UI-safe, no execution logic) |
| [Engine](./Engine/) | `@memberjunction/testing-engine` | Core test execution and evaluation engine supporting multiple test types |
| [CLI](./CLI/) | `@memberjunction/testing-cli` | Command-line interface for test execution and management |
