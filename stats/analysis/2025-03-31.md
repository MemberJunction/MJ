# Analysis — 2025-03-31

Hand-written source code (TypeScript + HTML + CSS) grew by roughly 5,650 lines over the quarter, a steady but unremarkable pace relative to prior periods. The headline story is instead in the supporting languages: Markdown nearly doubled from ~31K to ~56K lines (+25K), and JSON surged by nearly 74K lines — together those two categories account for the overwhelming majority of the quarter's total +134K-line delta. SQL also added ~24K lines, reflecting a sustained run of schema migrations accompanying active feature development.

The biggest feature driver was the MCP (Model Context Protocol) server for MemberJunction, introduced across PRs #865–#868 in the final weeks of the quarter. This work — MCP server plumbing, tool naming rules, and server improvements — was accompanied by substantial documentation and configuration files, which explains much of the Markdown surge. Running alongside it were deep CodeGen improvements: nullable-type fixes in generated entity classes (#861), advanced CHECK-constraint generation (#848, #858), physical column-order validation (#849), new entity-name rules (#859), and CodeGen GRANT statement tweaks (#830). The Action Engine received a significant revamp (#857, #854) and new unique-name enforcement (#860), while the communication providers gained message forwarding/reply support (#837), CC/BCC for MS Graph (#864), and related reply-field fixes (#847). Transaction management and sparse-update mutations for GraphQL (#824, #825) were also merged, and a new API-key authentication path was added (#736).

The JSON growth is almost entirely hand-written (+70K of the +74K total), pointing to metadata file expansion — new connector action metadata, configuration files for the new MCP server and AI agent schema additions (learning cycles, #720), and accumulated mj-sync declarative records rather than any single bulk import. The generated-vs-hand split is essentially unchanged at 46% for the source-code headline: the CodeGen machinery produced another +5K lines of entity classes and Angular forms, proportional to the new entities and field additions introduced by migrations this quarter. SQL remains entirely hand-written by the classification rules (the 446 generated SQL lines are a rounding artifact from a handful of CodeGen-run fragments), confirming that the +24K SQL growth reflects genuine schema investment: new tables for AI agent learning cycles, Query Entities, ConversationDetail columns, and the accumulated CodeGen-run migration files that formalize all of the above.
