# Analysis — 2024-12-31

Hand-written source code (TypeScript + JS + HTML + CSS + Markdown) grew by roughly 13,400 lines this quarter, reaching 90,587 — a moderate 17% increase over the 2024-09-30 baseline. The headline is that Markdown alone accounts for 8,038 of those lines, reflecting a significant documentation push across packages (READMEs for the vector, Rex recommendation, and duplicate-detection packages all landed via dedicated PRs #680–#683, #686). Strip the docs and the functional source growth is closer to 5,400 lines — deliberately paced, with the team consolidating and stabilizing rather than racing to ship new surface area.

The biggest technical driver on the TypeScript side (+3,898 hand-written lines) was a cluster of work across the communications, vector, and AI prompt subsystems. The `BaseCommunicationChanges` PR (#640) reworked core communication abstractions, while `AI-Prompt-config` (#657) and several `more-optimizations` passes (#650) refined prompt configuration and runtime. The vector layer saw dedicated optimization passes (#676, #677) and an updated Rex recommendation engine (#649, #658, #682). On the Explorer UI side, the quarter polished user-facing flows: an improved login screen (#642), a revamped new-user process (#645), user-view advanced tab (#635), add-single-record-to-list (#634), and a refreshed app-view create-record path (#685). The `unified-config` effort (#674) and `tagged-migration` (#678) each contributed a few hundred lines of infrastructure work, as did initial MJAPI Dockerization (#673, #696, #697).

The JSON surge (+38,525 total lines, almost entirely hand-written) tells a different story: this is declarative metadata growth, primarily `mj-sync` config files and connector action metadata added to `/metadata/integrations/`. It's large in line count but low in complexity — JSON arrays and nested objects expand quickly. The generated fraction of JSON actually shrank slightly (−256 lines), confirming that the new JSON isn't CodeGen output but authored configuration. Generated TypeScript grew by only +996 lines across the quarter, consistent with a period that saw few new entity migrations requiring full CodeGen sweeps; the generated-to-hand-written ratio in TypeScript stayed flat at roughly 2:1. The file count grew by 110 (1,662 → 1,772), reflecting new packages, README additions, and the metadata config files rather than any structural reorganization.
