Design notes — Option C

Visualizes the actual classify pipeline as nodes with their knobs. Click any stage to expand its config in the detail pane below. The branching arrow strip shows where each item ends up based on score / budget. Beautiful onboarding tool — explains the system and configures it in one view.

  • Strengths: teaches the architecture; "where does this knob fit in the flow?" is answered visually; the most demoable.
  • Weaknesses: expensive to build and maintain; ergonomically slower than B for repeated edits; needs custom layout logic for branching.
  • Best for: sales engineering; first-time admins; explaining the system to a stakeholder.
  • Distinctive: stage-as-node, knobs-as-pills inside each node; branching arrow strip with color-coded outcomes; selected-stage detail pane below.
